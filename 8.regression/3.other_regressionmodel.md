# 11.3 Other Regression Models
## Same Models, Different Objective

---

### 📖 Core Principle
> The models from classification (SVM, KNN, Decision Tree, Random Forest) can all be repurposed for regression. **The algorithm logic stays the same — only the output type, loss function, and final step change.**

---

## Classification vs Regression — What Changes

|                   | Classification                  | Regression |
|------------------ |---------------------------------|------------------|
| **Output type**   | Class label (discrete)          | Number (continuous) |
| **Loss function** | Cross-entropy / Gini            | MSE / MAE |
| **Final step**    | Predict class / apply threshold | Predict number directly |
| **Evaluation**    | Accuracy, F1, AUC               | MAE, RMSE, R² |
| **What stays same** | Core algorithm, feature handling, hyperparameter tuning ideas |

> 🔑 **"Same model. Different objective."**

---

## 1. Support Vector Regressor (SVR)

### 📖 Definition
> **SVR** doesn't find a boundary that separates classes — instead it finds a function that fits **within an epsilon-insensitive tube** around the data. Predictions within the tube incur zero penalty; only errors outside it are penalized.

### 🧑‍🏫 Layman Explanation
SVM (classifier) draws a line between two groups and maximizes the gap. 
SVR draws a line *through* the data and builds a **tolerance tube** around it. Any point inside the tube is "close enough" — it doesn't contribute to the error. Only the points outside the tube matter and determine the fit.

### ⚙️ Key Nuance
```
SVM Classifier:         SVR:
  * * | O O O          ~~~~~~~~~~~  ← upper boundary of tube
  * * | O O            - - - - - -  ← fitted function
  * * | O O O          ~~~~~~~~~~~  ← lower boundary of tube
  
  Maximize gap         Fit inside tube (epsilon-insensitive zone)
  between classes      Ignore errors within tube
```

**One-line intuition:**
- Classifier → Separates classes by maximizing margin
- Regressor → Fits values inside a tolerance zone

```python
from sklearn.svm import SVR
from sklearn.preprocessing import StandardScaler

scaler = StandardScaler()
X_s = scaler.fit_transform(X_train)   # SVR requires scaling!

svr = SVR(kernel='rbf', C=1.0, epsilon=0.1)
# epsilon = tube width (errors inside tube are ignored)
# C = penalty for errors outside tube
svr.fit(X_s, y_train)
```

## 2. KNN Regressor

### 📖 Definition
> **KNN Regressor** finds the K nearest neighbors to a query point and predicts the **mean of their target values** (instead of voting for a class).

### 🧑‍🏫 Layman Explanation
To predict a house's price, find the 5 most similar houses (nearest neighbors in feature space) and average their prices. Simple, but effective for local patterns.

### ⚙️ Key Nuance

|            | KNN Classifier                    | KNN Regressor |
|------------|-----------------------------------|------------|
| Prediction | **Majority class** of K neighbors | **Average value** of K neighbors |
| Output     | Class label                       | Continuous number |

```python
from sklearn.neighbors import KNeighborsRegressor
from sklearn.preprocessing import StandardScaler

scaler = StandardScaler()
X_s = scaler.fit_transform(X_train)   # Must scale!

knn_r = KNeighborsRegressor(n_neighbors=5)
knn_r.fit(X_s, y_train)

y_pred = knn_r.predict(scaler.transform(X_test))
# Prediction = average of 5 nearest neighbors' target values
```

**One-line intuition:**
- Classifier → Assigns the most common neighbor class
- Regressor → Predicts the average of neighbor values

## 3. Decision Tree Regressor

### 📖 Definition
> **Decision Tree Regressor** uses the same if-else splitting structure as the classifier, but **leaf nodes store the average of target values** (not a class label). Splits are chosen to **minimize variance** (not impurity/Gini).

### ⚙️ Key Nuance

|                     | Decision Tree Classifier           | Decision Tree Regressor |
|---------------------|------------------------------------|--------------------------|
| Leaf stores         | Class label                        | **Average numeric value** |
| Splitting criterion | Gini / Entropy (minimize impurity) | **Variance reduction** (MSE) |
| Output              | Class                              | Continuous number |

```
Tree example:
         X₁ < 20?
         /       \
     Yes           No
      │              │
   X₂ < 200?    X₂ < 170?
   /      \       /       \
300.5    65.7  [X₃ < 40?] 1023
               /        \
           -64.1        0.7
←— Leaf nodes store average target values, not class labels —→
```

```python
from sklearn.tree import DecisionTreeRegressor

dtr = DecisionTreeRegressor(max_depth=5, min_samples_split=10)
dtr.fit(X_train, y_train)

y_pred = dtr.predict(X_test)
# Each prediction = average target value of the leaf node the sample lands in
```

**One-line intuition:**
- Classifier → Assigns classes using if-else rules
- Regressor → Predicts values by averaging leaf outputs

## 4. Random Forest Regressor

### 📖 Definition
> **Random Forest Regressor** builds many Decision Tree Regressors on bootstrap samples, each predicting a number, and the **final output is the average across all trees** (not majority vote).

### ⚙️ Key Nuance

|                   | Random Forest Classifier        | Random Forest Regressor |
|-------------------|---------------------------------|-------------------|
| Each tree outputs | Class label                     | Numeric value |
| Aggregation       | Majority vote                   | **Average of all trees** |
| Benefit           | Reduces variance vs single tree | Same, but for numeric prediction |

```python
from sklearn.ensemble import RandomForestRegressor
from sklearn.metrics import mean_absolute_error, r2_score

rfr = RandomForestRegressor(n_estimators=100, max_depth=6, random_state=42)
rfr.fit(X_train, y_train)

y_pred = rfr.predict(X_test)
# Final prediction = mean of 100 individual tree predictions

print(f"MAE:  {mean_absolute_error(y_test, y_pred):.2f}")
print(f"R²:   {r2_score(y_test, y_pred):.4f}")

# Feature importance — same as classifier, very reliable
import pandas as pd
feat_imp = pd.Series(rfr.feature_importances_).sort_values(ascending=False)
print(feat_imp.head())
```

**One-line intuition:**
- Classifier → Predicts classes using majority voting
- Regressor → Predicts values by averaging all trees

## ⭐ Regressor Nuance Summary

| Model         | Classifier output          | Regressor output         | Key change |
|---------------|----------------------------|--------------------------|---------------|
| SVR           | Max margin boundary        | Fits inside epsilon-tube | Margin → tolerance zone |
| KNN           | Majority class vote        | Average of K values      | Vote → mean |
| Decision Tree | Class at leaf              | Average value at leaf    | Label → mean; Gini → Variance |
| Random Forest | Majority vote across trees | Average across trees     | Vote → mean |
