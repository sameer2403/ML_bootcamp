# 11.1 Regression — Fundamentals

## 📖 Definition
> **Regression** is a type of **Supervised Learning** where the model learns a mapping from input features to a **continuous numerical output** (a real number, not a class label).

---

## 🧑‍🏫 Layman Explanation
Classification answers "which bucket?" — Regression answers "how much?" Predicting whether someone will buy (Yes/No) is classification. Predicting how much they'll spend (₹2,340) is regression. The model learns the relationship between inputs and a target number, then uses that relationship to predict new numbers it has never seen.

---

## ⚙️ Technical Explanation

|                | **Classification**    | **Regression** |
|----------------|-----------------------|-----------------|
| Output.        | Discrete class label  | Continuous number |
| Example output | Spam / Not Spam       | ₹45,00,000 house price |
| Goal           | Find decision boundary | Find best-fit function |
| Loss function  | Cross-entropy         | MSE / MAE |
| Evaluation     | Accuracy, F1, AUC     | MAE, RMSE, R² |

**How regression models think:**
1. **Learn Patterns** — how do features (size, location, age) affect the target (price)?
2. **Map Inputs** — build a function `f(X) → ŷ` (a continuous number)
3. **Minimize Error** — adjust the function so predictions are as close to actuals as possible

---

## 💡 Common Use Cases

| Problem                | Input Features                     | Continuous Output |
|------------------------|------------------------------------|-------------------|
| House Price Prediction | sqft, bedrooms, location           | ₹ price |
| Salary Prediction      | years of experience, role, company | ₹ salary |
| Sales Prediction       | ad spend, season, region           | units sold |
| Insurance Pricing      | age, health history, location      | premium amount |
| Demand Prediction      | price, weather, day of week        | units demanded |
| Risk Score Prediction  | transaction history, geography     | risk score 0–100 |

---

## ⭐ Key Highlights
> 🔑 **Output is always a number, not a category**
> 🔑 **Accuracy is NOT used for regression** — error-based metrics instead (MAE, RMSE, R²)
> 🔑 Goal is to **minimize prediction error**, not find a boundary

---

## 🔄 Regression Workflow
```
Data            Model           Prediction       Evaluate        Optimize

Clean           Regressor       test data
```
