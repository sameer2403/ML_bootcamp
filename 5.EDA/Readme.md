## What Is EDA?

# Definition

Exploratory Data Analysis (EDA) is the process of exploring,
understanding, and summarizing data before building models.

-> Explore
-> Understand
-> Summarize

---------------------------------------------------------------------------
## Why EDA Matters?

-> Understand Behavior
See patterns, trends, and structure.

-> Decide Preprocessing Steps
Choose cleaning and transformations.

-> Detect Issues
Identify missing values, noise, and errors.

-> Improve Performance
Better data produces better ML models.



---------------------------------------------------------------------------

## Types of Features

# Numerical
Discrete (or) Continuous
Example: No. of Products sold; Weight

# Categorical
Nominal (or) Ordinal
Example: Gender; Size (S, M, L)

# Datetime
Time related data
Example: Date, Month, Timestamp

# Text
Unstructured Natural Language
Example: News, Reviews, Messages
